Executive Summary

핵심 인사이트

인터뷰 2건 + 본인 서사 1건, 연속 1,000회 돌파. 채널 정체성이 "퇴사/전환 서사 플랫폼"으로 형성 중.

전략 방향

4개 콘텐츠 축 운영. 터지는 콘텐츠(인터뷰/본인서사)에 집중 투자, 실험 콘텐츠(AI챌린지)는 압축 완주.

기대 효과

에너지 효율화 + 구독자 성장 가속 + 비즈니스(뉴리치 클래스) 연결 기반 구축

1. 현황 분석

1.1 채널 기본 지표

지표 수치 비고
구독자 2,260명 -
평균 조회수 약 200회 일반 영상 기준
업로드 빈도 주 2회 화/토 본편 + 평일 Shorts

1.2 콘텐츠별 성과 비교

콘텐츠 유형 조회수 상태 평가
인터뷰 A 1,000+ 터짐 평균 5배
인터뷰 B 1,000+ 터짐 평균 5배
본인 서사 (삼성 퇴사) 1,000+ 터짐 평균 5배
AI 챌린지 (3개 업로드) 100~555 평균 이하 시리즈 한계
나눔 (마진계산기) 156 참여율 7.7% DB 수집용
핵심 발견

시청자는 "인터뷰이"만 보러 온 게 아님. 본인 서사도 1,000회 돌파 → 박과장 자체에 대한 관심 확인

2. 전략 방향

2.1 채널 포지셔닝

항목 정의
채널 정체성 "퇴사/전환 서사 플랫폼"
핵심 메시지 퇴사 후 삶, 실제로 어떻게 굴러가는가
타겟 시청자 퇴사/이직/창업 고민 중인 30~50대 직장인
차별화 포인트 전문가가 아닌 "같이 고민하는 동료" 포지션

2.2 콘텐츠 4개 축 정의

축 역할 빈도 목적 에너지
인터뷰 메인 격주 1회 조회수, 구독자 확보 ⭐⭐⭐⭐
본인 서사 서브 메인 월 2~3회 채널 정체성 확립 ⭐⭐⭐
AI 챌린지 시즌제 남은 분량 완주 실험/화제성 ⭐⭐
나눔 부스터 월 1회 DB 수집, 신뢰 구축 ⭐
핵심 원칙

터지는 콘텐츠(인터뷰/본인서사)에 70% 에너지 집중. 나머지는 효율적으로 운영.

3. 콘텐츠별 실행 계획

1 인터뷰 시리즈

항목 내용
시리즈명 50+ 퇴사 프로젝트
빈도 격주 1회
인트로 포맷 "저는 18년 직장생활 접고 지금 3개 사업 운영 중인 박과장입니다" (5초)
엔딩 CTA "비슷한 고민 중이시면 구독 → 다음 이야기 먼저 보실 수 있습니다"

썸네일/타이틀 방향

  • 숫자 + 감정 조합: "40대 가장, 3년 모은 돈 다 쓰고 떠납니다"
  • 정보성 + 호기심: "독일 이민 비용 공개. 이 정도면 갈 수 있나요?"

2 본인 서사 시리즈

항목 내용
빈도 월 2~3회
목적 채널 정체성 확립, 신뢰 구축

콘텐츠 아이디어

  • 삼성 18년, 왜 떠났나 (후속편)
  • 퇴사 후 첫 1년, 수입 공개
  • 3개 사업 동시 운영, 하루 루틴
  • 퇴사 전 재정 어떻게 준비했나
  • 40대 퇴사, 가족은 뭐라 했나

3 AI 챌린지 (압축 운영)

항목 기존 변경
남은 분량 본편 5개 본편 2개 + Shorts 3개
촬영 횟수 5회 2회

변경된 구성

회차 내용 형식
4화 중간 점검 (현재 매출/상황) 본편
5~7 과정 하이라이트 Shorts 3개
8화 최종 결과 공개 본편 (집중 투자)

결과 영상 타이틀 (사전 설계)

결과 타이틀 방향
성공 "ChatGPT가 99% 실패 예측한 상품, 8주 만에 ___만원"
실패 "AI 말 듣고 8주 도전, 결국 망했습니다 (쓴 돈 공개)"
애매 "99% 실패 상품 8주 결과, 본전은 뽑았을까?"

4 나눔 콘텐츠

항목 내용
빈도 월 1회
형식 Shorts + 설명란 링크 (효율적)
목적 DB 수집, 뉴리치 클래스 잠재고객 확보
현재 성과 조회 156 / 댓글 12건 (참여율 7.7%)

나눔 아이템 후보

  • 마진계산기 (완료)
  • 퇴사 재정 체크리스트
  • 공동구매 계약서 템플릿
  • 사업자등록 준비 가이드
